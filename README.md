# WarCardGame

A simple iOS card game built using SwiftUI that simulates a round-based version of the classic card game War. 
The player and CPU each draw a random card, and the higher card wins the round.

# Features
- SwiftUI-based UI layout
- Randomized card dealing
- Score tracking for Player vs CPU
- Simple game logic with real-time UI updates
- card and background images
